这是一个《街头霸王6》训练模式辅助脚本模组，用于在训练场中快速设定或随机化体力、斗气、超必杀槽、角色专属资源、起始位置和游戏速度，方便复现更接近实战的复杂局面。

## 中文字体

中文 UI 由 `TrainingModePlusScripts/Localization.lua` 负责加载字体。

如果中文显示为方块，说明字体文件没有正确加载。请确认 `reframework/fonts/TrainingModePlusCN.ttf` 存在，并且该字体支持简体中文。

## 主窗口按钮

`刷新训练模式`：请求游戏重新应用训练模式参数。通常用于手动刷新当前设置。

`刷新模块`：重新初始化脚本模块。如果界面或参数出现异常，可以点击此按钮恢复模块状态。

`刷新并随机化`：按照当前启用的随机化配置生成一组新参数，并刷新训练场。

默认热键为：

```text
Select + L3
```

热键右侧可以重新设置按键。右键点击热键控件可打开选项，包括清除、重置为默认、启用组合键等。

注意：当前版本中，使用热键触发 `刷新并随机化` 时，会额外执行一次模块刷新，用于减少随机化后状态不同步的问题。

## 体力

路径：

```text
训练设置 + 随机化 > 体力 > 玩家1 体力 / 玩家2 体力
```

可用功能：

- 直接设置体力百分比。
- 启用体力随机化。
- 添加多个体力配置。
- 每个配置可设置体力下限、体力上限和触发权重。
- 勾选 `统一触发频率` 后，所有启用的体力配置按相同概率触发。

示例：

```text
体力配置 1：80 - 100
体力配置 2：30 - 50
统一触发频率：关闭
配置 1 权重：7
配置 2 权重：3
```

表示随机化时约 70% 概率得到高血量，约 30% 概率得到低血量。

## 斗气

路径：

```text
训练设置 + 随机化 > 斗气 > 玩家1 斗气 / 玩家2 斗气
```

可用功能：

- 设置斗气库存。
- 设置是否斗气枯竭。
- 启用斗气随机化。
- 添加多个斗气配置。
- 斗气配置值范围为 `-6` 到 `6`。

斗气配置值说明：

- `0` 到 `6`：正常斗气库存。
- 负数：进入斗气枯竭状态，绝对值表示枯竭后的斗气量。

勾选 `统一触发频率` 后，所有启用的斗气配置按相同概率触发。

## 超必杀槽

路径：

```text
训练设置 + 随机化 > 超必杀槽 > 玩家1 超必杀槽 / 玩家2 超必杀槽
```

可用功能：

- 按整格设置超必杀槽，范围为 `0` 到 `3`。
- 切换超必杀类型：整格 / 任意格数。
- 使用任意格数模式时，可选择点数输入或小数格输入。
- 启用超必杀槽随机化。
- 勾选 `统一设定超必杀槽随机化范围` 后，可设置随机下限和上限。

## 角色专属资源

路径：

```text
训练设置 + 随机化 > 角色专属资源
```

脚本会根据当前双方角色显示可用的角色专属资源。当前已包含：

- 隆：电刃蓄力
- 杰米：魔身、醉酒等级
- 金伯莉：手里剑炸弹库存
- 莉莉：风缠库存
- 蛛俐：风水引擎、风破库存
- 本田：相扑魂
- 布兰卡：雷兽、小布兰卡炸弹
- 古烈：坚实打击者
- 不知火舞：火焰库存
- C.毒蛇：限制解除
- 英格丽德：太阳纹章
- 玛侬：奖牌等级

支持的功能取决于角色资源类型：

- 计时类资源：可设置标准、发动、无限，部分资源可设置起始发动值。
- 库存类资源：可设置库存数，部分资源可启用无限。
- 可对资源启用随机化，并设置随机范围。

## 位置参数

路径：

```text
训练设置 + 随机化 > 位置参数
```

先勾选 `启用起始位置调整` 才会显示完整位置配置。

通用设置：

- `使用离散位置值`：用整数参考值表示屏幕位置，适合快速配置。
- `统一设定起始位置`：所有位置配置共用同一个起始位置。
- `使用预设相对距离`：用预设距离代替手动数值。
- `统一设定相对距离`：所有位置配置共用同一个相对距离。
- `统一设定玩家站位侧`：所有位置配置共用同一个左右站位规则。
- `统一触发频率`：所有启用的位置配置按相同概率触发。

预设相对距离包括：

- 贴身
- 近距离
- 中距离
- 远距离
- 拆投距离
- 开局距离
- 牵制距离
- 最大距离

启用 `使用预设相对距离` 时，下拉选项会直接显示预设名称和当前相对距离数值。

自定义位置配置：

- 每个 `位置配置` 都是一个可随机抽取的起始局面。
- 配置内可设置起始位置、相对距离、玩家站位侧和触发权重。
- 玩家站位侧可选随机、玩家在左侧、玩家在右侧。

## 游戏速度

路径：

```text
游戏速度 Plus
```

可设置游戏速度：

```text
50% / 60% / 70% / 80% / 90% / 100% / 110% / 120% / 130% / 140% / 150%
```

`重置为标准` 会恢复为 100%。

脚本菜单和游戏内训练模式菜单可以同时使用，最后一次修改会生效。

- 体力

## 配置保存

训练设置会保存到：

```text
TrainingModePlus/TrainingSettingsAndRandomizer_Config.json
```

离开训练模式时会自动保存当前配置。下次进入训练模式时会尝试读取并恢复。

## 常见问题

### 中文显示为方块

确认 `reframework/fonts/TrainingModePlusCN.ttf` 存在，并且字体支持中文。替换字体后重启游戏。

### 随机化后参数没有立刻生效

点击 `刷新训练模式`。如果仍异常，点击 `刷新模块`。

使用默认热键触发 `刷新并随机化` 时，脚本已经会自动请求模块刷新。

### 右键热键菜单仍有英文

当前模组已对常见 Hotkeys 文本做了本地化包装。如果出现新的英文提示，需要在 `Localization.lua` 的 `hotkey_text` 表中继续添加翻译。

## 建议用法

体力训练：

- 玩家1 固定 100%。
- 玩家2 开启体力随机化。
- 设置多个配置，例如 100%、80%、50%、30%。

斗气确认：

- 开启双方斗气随机化。
- 为正常斗气和枯竭状态分别添加配置。

位置复现：

- 开启起始位置调整。
- 使用多个位置配置，分别设置贴身、拆投距离、开局距离和版边附近位置。
- 关闭 `统一触发频率` 时，可用权重控制高频局面和低频局面。

综合实战随机：

- 同时开启体力、斗气、超必杀槽、位置随机化。
- 使用热键 `Select + L3` 快速刷新并随机化。
